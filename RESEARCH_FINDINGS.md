# Research Findings — Chase Perks Maximization (April 2026)

Curated from points blogs, FlyerTalk, churning communities, and official Chase docs. Used to enrich the agent's decision rules, knowledge base, and weekly recommendations.

## 1. Major 2026 changes the agent must know

- **Annual fee jumped to $795** (from $550). Justification math for the agent: must surface ≥$795 of captured value/year or the user is losing money.
- **Both Sapphire allowed simultaneously** since Jan 22, 2026. Agent could later suggest adding CSP if user is under 5/24 — but only if welcome bonus + transfer partner value clearly justifies.
- **Points Boost replaces fixed 1.5 cpp**. Baseline portal value now 1 cpp; Points Boost lifts to 1.5–2 cpp on rotating offers. Existing CSR holders keep 1.5 cpp until Oct 26, 2027 (transition window).
- **Travel earning devalued**: 4x flights/hotels booked direct, 1x on Uber/transit/OTAs (was 3x on all travel).

## 2. The killer 2026 stack: triple hotel credit (worth up to $800)

The single biggest opportunity the agent should push hard if user has any 2+ night stay in 2026:

- **$300 annual travel credit** (anniversary year)
- **$500 The Edit credit** (calendar year, 2× $250 max per transaction, 2+ night prepaid)
- **$250 select Chase Travel hotel credit** (2026 only, prepaid, 2+ nights at: IHG, Montage, Pendry, Omni, Virgin Hotels, Minor, Pan Pacific)

Ideal target: a property that is **both in The Edit AND in the eligible brand list** → triggers Edit + select-hotel credits on the same booking. Pendry (Park City, Manhattan West), some Virgin/Omni properties.

Booking rules:
- Must be "Pay Now" prepaid
- 2+ consecutive nights
- Total cost ≥ $500 to capture both $250 credits
- Cancellation reverses the credit
- Edit credits cannot stack on the SAME booking — must be 2 separate stays
- IHG One Rewards Platinum status is complimentary (activate via Chase) → 60% bonus points + space-available upgrades

Order credits apply: travel credit first → select-hotel → Edit.

## 3. DoorDash credit reality check (April 2026)

- **$20 minimum order now applies on convenience pickups** (DoorDash explicitly added this to kill the workaround)
- The "free pistachios at 7-Eleven" trick is mostly dead, though some local stores still allow sub-$20 orders
- $5 restaurant + 2× $10 non-restaurant credits per month, doesn't roll over
- **Stack store-specific offers** (under "Deals and gift cards") with the Chase promo
- **Discount applies to subtotal only** — not fees/taxes/tip
- Alcohol workaround killed (state limits)
- Strategy: target subtotal at exactly $20 on stuff you'd buy anyway (groceries, household). Set last-week-of-month reminder.
- Households with 2 CSR cards can split DoorDash accounts → $600/yr promo value.

→ Agent decision rule: this credit's "real value" is now ~$10–15/month not $25. Discount the headline value when computing missed-value.

## 4. Lyft credit — NYC specifics

- **$10/month, applies to subtotal directly** in-app (not statement credit)
- Set CSR as default in Lyft → must NOT pay via Apple Pay/Google Pay/Venmo (disqualifies)
- **No Wait & Save rides, no bikes/scooters** (not eligible)
- **5x points on all Lyft rides** with CSR (vs 1x on Uber under new rules)
- Single NYC ride usually >$10, so credit consumed in one trip
- DashPass linkage gives extra Lyft discount → stack
- → Agent rule update for Lyft vs Uber: Lyft now wins more often than the spec implies. Compare with: `(Uber price + 1x points value) vs (Lyft price - $10 credit + 5x points value)`. Lyft typically wins unless Uber is >$15 cheaper.

## 5. Dining credit — NYC venue intel

- ~55 NYC restaurants in Sapphire Reserve Exclusive Tables on OpenTable
- Each $150 half-year credit → can be hit in a single upscale meal
- Standout NYC venues: Gramercy Tavern, Estela, Tolo, Una Pizza Napoletana, Bar Kabawa, Kabawa ($145 prix fixe = perfect credit-burn), Le B, Fairfax
- **inKind hack**: Some venues (Gjelina) are on inKind AND eligible — split bill across both
- Use third-party maps (Nextcard MealMaxxer, Use Your Credits) to filter by neighborhood/cuisine — Chase doesn't publish a usable list
- → Agent decision rule: in May/June and Nov/Dec, push aggressive booking suggestion. Target a single $150 meal rather than 3× $50 (lower friction, more enjoyable, less risk of activation issues).

## 6. StubHub credit — booking date is what matters

- **Trigger date = purchase posting date, not event date** → can buy December tickets for events months later
- Activation required (Chase app → "$300 StubHub Credit" tile)
- Taxes & fees included in credit calc
- Gift cards excluded as triggers, but **buying StubHub gift cards on sale and then applying them** to a larger purchase = lever to maximize cash value
- Viagogo works (US site only — viagogo.com, NOT stubhub.co.uk)
- Resell strategy: buy → resell on StubHub → recover 85–90% after 10–15% seller fees
- → Agent rule: in late June and late December, push StubHub use. NYC events the agent should be aware of as candidates: Knicks, Rangers, Yankees, Mets, Broadway, MSG concerts, Beacon Theatre, comedy.

## 7. JFK T4 Sapphire Lounge — waitlist reality

- Lounge is small (7,600 sqft) and overcrowded due to Etihad premium pax
- **Real wait times: 1 hour+ during peak**
- Strategy: **arrive 2.5–3 hr early, get on digital waitlist immediately, then explore alternatives** (text notification when ready)
- Capital One Lounge at JFK is reported as superior with no wait — but requires Capital One card
- → Agent rule: for JFK T4, recommend Sapphire Lounge BUT with explicit "join waitlist immediately" instruction. Mention Primeclass/KAL as backups.

## 8. LGA Terminal B Sapphire Lounge — under-hyped jewel

- Widely rated **best Priority Pass lounge in the US**
- 21,800 sqft across 2 floors, complimentary 20-min facials, full bar, Joseph Leonard food, showers
- Open 4:30am–9:30pm
- 3-hour pre-flight access window
- Departures-only (cannot enter on arrival boarding pass)
- → Agent rule: if user has LGA Terminal B departure (American/JetBlue/Air Canada/Southwest/United), this is a HIGH-priority recommendation. Mention spa appointment.

## 9. Apple TV+/Music — Family Sharing leverage

- Active subscriptions auto-suspend on activation
- Worth ~$288/year combined
- **Family Sharing works for Apple TV+** (extends to family members)
- **Apple Music is individual plan only** — Family Sharing limited; can't be upgraded to Family without paying full
- → Agent rule: one-time activation suggestion, then silent unless missed. Mention Family Sharing for Apple TV+ specifically if user has family.

## 10. Peloton credit — the standalone hack

- **No Peloton equipment needed** — credit applies to standalone Strength+ app ($9.99/mo)
- $10/month credit fully covers Strength+ → essentially free fitness app
- **Must subscribe directly via Peloton (not App Store)** and pay with CSR directly (not Apple Pay)
- Activation required via Chase app
- → Agent rule: one-time push to activate Strength+ if user isn't using Peloton. After activation, fully passive — never remind again unless billing breaks.

## 11. Travel credit — "everything counts"

- Far broader than Amex Platinum airline credit
- Eligible: airlines (incl. baggage/seat fees), hotels, car rentals, cruises, tolls, parking, taxis, Uber/Lyft, Amtrak, buses, OTAs (Expedia, Kayak), campgrounds
- $300 cap can be split across multiple purchases
- Reset on **anniversary year** (not calendar)
- Returns within 90 days reverse the credit
- → Agent rule: this credit auto-burns in NYC just from Uber + the occasional Amtrak/parking. Track silently; only flag if user is ≥10 months into anniversary year with <$200 used.

## 12. Hyatt is the points sweet spot

- 2.05 cpp Chase value vs 1.7 cpp Hyatt value → favorable transfer
- Cat 1–4 free nights: 3,500–8,000 pts, real cash $250–$650 → **4.8–6.2 cpp**
- Ziva/Zilara all-inclusive: 25k–35k pts vs $750+ rooms → **3+ cpp** with food/drinks included
- Max award night cap: 45,000 pts (vs Marriott/Hilton charging 100k+ for similar properties)
- **WARN: new Hyatt award chart May 2026** → peak pricing increases. Lock in stays before then.

## 13. Other transfer partner sweet spots

- **Air Canada Aeroplan**: stopovers in Europe for ~70k pts one-way → 5–6 cpp on premium cabins
- **Virgin Atlantic Flying Club**: Delta One/ANA First → 5.5–6.5 cpp (best long-haul value)
- **Iberia Avios**: BOS/JFK → MAD off-peak from 34k Avios one-way → 4.7 cpp
- **Avoid**: Marriott Bonvoy, Wyndham (sub-1 cpp typical) — only use Wyndham during transfer bonuses (March 2026 had first-ever 30%)
- **Watch transfer bonuses**: Chase typically runs 20–30% airline / 50–80% hotel bonuses

## 14. Chase Private Client — real ROI levers

Watch for: J.P. Morgan Private Client now charges fees if balance <$750k as of 2026.

## 15. Revolut Metal vs CSR — by category

| Category | Winner | Reason |
|---|---|---|
| US-domestic spend | CSR | 3x dining, 4x direct travel, 1x base |
| EU/UK weekday FX | Revolut Metal | Interbank rate, no Visa markup |
| Weekend FX | CSR | Revolut adds weekend markup |
| Multi-currency wallet / holding EUR | Revolut | Native multi-currency |
| Travel insurance / trip protection | CSR | Strong protections, Revolut weak |
| Foreign ATM withdrawals | Revolut Metal | Higher fee-free limits |
| Lounge access | CSR (PP + Sapphire) | Revolut SmartDelay weaker |
| Online foreign merchant | CSR | $0 FX + 1x points + protection |

→ Agent rule: default to CSR abroad UNLESS user is exchanging large sums or operating during weekends in volatile currencies.

- **Pop-up jail** still active → triggered by churning patterns / low spend on existing Chase

## 20. Behavior gaps in the original spec to address

1. Spec doesn't mention **anniversary year vs calendar year** — agent must track both clocks separately and never confuse them.
2. Spec doesn't model **transfer partner recommendations** — should add a "redemption advisor" mode (which partner to transfer to for a given trip).
3. Spec doesn't surface **travel insurance reminders** — when a flight is booked on CSR, agent should note coverage.
4. Spec doesn't track **"active subscription" state** with last-verified date — needed because Chase activations sometimes break silently.
5. Spec doesn't mention **2026-only $250 select hotels credit** — must add explicitly with deadline (12/31/26).
6. Spec doesn't mention **IHG Platinum activation** — easy free win.
7. Spec implies "DashPass = $120 value" — should reflect that real captured value depends on usage frequency; flag if user has DashPass but rarely orders.
8. Spec doesn't address **5/24 status** for CPC card-acquisition advice — relevant if user wants to add cards.
9. Spec doesn't track **redemption dates** for sign-up bonus 90-day window.
10. Spec doesn't handle **cancellation reversal** — if a hotel is cancelled, agent must know the credit reverses.
